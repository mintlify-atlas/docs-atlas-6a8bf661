    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your sentiment analysis API running in minutes
  </p>

  <Steps>
    <Step title="Install dependencies">
      Clone the repository and install the required Python packages.

      ```bash
      git clone https://github.com/santiicristobal/smart-feedback-api.git
      cd smart-feedback-api
      pip install -r requirements.txt
      ```
    </Step>

    <Step title="Start the server">
      Launch the FastAPI server with Uvicorn.

      ```bash
      uvicorn app.main:app --reload
      ```

      The API will be available at `http://127.0.0.1:8000`
    </Step>

    <Step title="Make your first request">
      Send a POST request to analyze sentiment.

      ```bash cURL
      curl -X POST "http://127.0.0.1:8000/analyze" \
        -H "Content-Type: application/json" \
        -d '{"text": "El servicio fue excelente, muy satisfecho"}'
      ```

      <Accordion title="Example Response">
        ```json
        {
          "sentiment": "positivo",
          "score": 0.9234567
        }
        ```
      </Accordion>
    </Step>

    <Step title="Explore the API">
      Visit the auto-generated API documentation at `http://127.0.0.1:8000/docs` to test endpoints interactively.
    </Step>
  </Steps>
</div>
