## What is Smart Feedback API?

Smart Feedback API is a REST API that analyzes sentiment in user feedback using machine learning. You send text to the API, and it returns the detected sentiment (positive, neutral, or negative) along with a confidence score.

The API is built with FastAPI and uses scikit-learn for sentiment classification, making it fast, reliable, and easy to integrate into your applications.

## Technology stack

The Smart Feedback API is built with modern Python technologies:

- **FastAPI** - Fast, modern web framework for building APIs
- **scikit-learn** - Machine learning library for model training and prediction
- **pandas** - Data manipulation and analysis
- **Uvicorn** - Lightning-fast ASGI server
- **Pydantic** - Data validation using Python type annotations

## How it works

The API follows a simple three-step process:

1. **Text preprocessing** - Your input text is cleaned and normalized using `app/preprocessing.py:4`
2. **Vectorization** - Text is converted to numerical features using TF-IDF
3. **Classification** - A Logistic Regression model predicts sentiment with confidence scores

The model returns probabilities for each sentiment class (positive, neutral, negative) and selects the one with the highest confidence as the final prediction.

## Use cases

Smart Feedback API is ideal for:

- **Customer feedback analysis** - Automatically categorize product reviews and support tickets
- **Social media monitoring** - Track sentiment in user comments and posts
- **Survey analysis** - Analyze open-ended survey responses at scale
- **Quality assurance** - Monitor sentiment trends in user feedback over time

## API response format

All sentiment analysis requests return a consistent response format defined in `app/main.py:22`:

```json
{
  "sentiment": "positivo",
  "score": 0.9498749406711353
}
```

- `sentiment` - The predicted sentiment class (positivo, neutral, or negativo)
- `score` - Model confidence for the prediction (0.0 to 1.0)
